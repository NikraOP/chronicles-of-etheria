# Хроники Этерии — команда агентов

Браузерная RPG: **vanilla JS**, без сборщика. Скрипты в `index.html` в строгом порядке. Сохранение: `localStorage`, ключ `rpg_save_v21`.

## Когда кого вызывать

| Задача | Субагент (`name`) | Пример запроса |
|--------|-------------------|----------------|
| Баг, ошибка, сломанная механика | `охотник-за-багами` | «После боя HP не восстанавливается» |
| Сложность, статы, монстры, цены | `балансировщик` | «Орки на 8 ур. убивают за 2 хода» |
| Новая локация, монстр, способность, предмет в БД | `мастер-контента` | «Добавь локацию Болото, 20–25 ур.» |
| Логика боя, эффекты, ходы, способности в коде | `инженер-боя` | «Добавь отражение урона в бой» |
| Вёрстка, CSS, экраны, анимации | `художник-интерфейса` | «Инвентарь на телефоне обрезается» |
| Сейв, импорт/экспорт, миграция | `хранитель-сохранений` | «Старый сейв не грузится» |
| Магазин, крафт, профессии, ресурсы | `экономист` | «Крафт меча слишком дорогой» |
| Способности монстров, дебаффы, AI боя | `инспектор-монстров` | «Паутина сразу пропадает» |
| Неясно, с чего начать | `координатор-этерии` | «Хочу новую ветку квестов» |

Вызов: `Use the <name> subagent: …` или «делегируй субагенту балансировщик».

## Архитектура кода

```
js/data/          — константы (LOCATIONS, EQUIPMENT_DB, ABILITIES_DB, …)
js/core/          — логика (player, battle/, inventory, shop, crafting, gathering)
js/ui/            — render, navigation, animations
js/main.js        — init, сейвы, скины, миграции
css/style.css     — стили
```

## Правила для всех агентов

- Минимальный diff; не рефакторить без запроса.
- Классы: `Воин`, `Маг`, `Разбойник` — строки на русском.
- Не коммитить и не пушить без явной просьбы.
- Баланс не трогать в багфиксах; баги не чинить в задачах баланса — делегировать.

Подробные промпты: `.cursor/agents/*.md`
