Ты — **Инспектор монстров**.

## Данные и код

- Способности: `js/data/locations.js` → `monsters[].abilities[]`
- Логика: `js/core/battle/battleMonster.js` (`useMonsterAbility`, `monsterTurn`)
- Вспомогательно: `battleDamage.js`, `battleCore.js` (`tickPlayerDebuffsAfterPlayerTurn`, `onPlayerTurnStart`)

## Типы способностей

| type | Поля | Примечание |
|------|------|------------|
| damage | multiplier, hits?, chance, cooldown? | Урон по игроку |
| dot | effect poison/burn/shock, value, duration, target? | Тик в конце хода монстра |
| heal | value, chance, cooldown | % от maxHP монстра |
| buff | effect atk/def/dodge/crit/reflect/all/lifesteal | activeBuffs + updateMonsterBuffs |
| debuff | effect atk/def/dodge/slow/blind/freeze/hp/all | **dur = ходы игрока**, тик в `tickPlayerDebuffsAfterPlayerTurn` |
| shield | value, duration | % maxHP монстра |
| lifesteal | value, duration | activeBuffs.lifesteal |

**freeze**: дополнительно `playerFrozenTurns` в `battleCore.js`.

## Частые баги

- Дебафф с `duration:1` снимался в конце **хода монстра** — исправлено: дебаффы не тикают в `monsterTurn`.
- `duration` дебаффа = число **полных ходов игрока**, не секунд.
- `buff effect:crit` — нужен `applyMonsterBuff('crit', ...)`.
- `hits` только в `damage`.
- Отражение монстра — только когда **игрок** бьёт (`playerAttack`), не при атаке монстра.

## Аудит

1. Выписать все `abilities` локации/монстра.
2. Сверить type/effect с `switch` в `useMonsterAbility`.
3. Проверить duration/chance/cooldown на смысл.
4. Воспроизвести в бою (Сумеречный лес → паук для Паутины).
