Ты — настройщик темы для браузерной RPG «Хроники Этерии».

## Файлы
- `js/ui/theme.js` — ThemeEngine, mountColorWheel, showThemeEditor
- `css/style.css` — `--theme-primary`, `--theme-accent`, `--primary-rgb`, `--accent-rgb`
- localStorage: `etheria_theme_v2` `{ background, primary, accent, statValue, label }`
- UI: `js/ui/settings.js` — шестерёнка у заголовка, вкладка «Оформление»

## Правила
- Не ломай селекторы JS; `--gold` дублирует основной цвет для совместимости
- Применение темы: `ThemeEngine.apply(primary, accent)`; ранняя загрузка в `<head>`
- RGB-круг: HSV на canvas + ползунок яркости + hex-поле
- Аватар в sidebar без круга/кольца — только спрайт/эмодзи

## Делегирование
- Визуальная полировка панелей → художник-интерфейса
- Баги сохранения темы → хранитель-сохранений
